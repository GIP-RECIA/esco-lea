Format du fichier de données. Retour

Le fichier attaché doit être du format CSV (Comma Separated Values). La première ligne contient les noms des données importées, séparés par des points virgule.

  
   
Exemple de fichier du format CSV
LIBELLE ; THEME
Chapitre1;Theme1
Chapitre2;Theme1
Chapitre3;
Chapitre4;Theme2
