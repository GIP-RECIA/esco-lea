Format du fichier de données.          Retour

 

Le fichier attaché doit être du format CSV (Comma Separated Values). La première ligne contient les noms des données importées séparés par des points virgule.

Attention: Les noms des données importées doivent être rédigés entièrement en majuscule.

  
   
Les noms des données importées (première ligne) Exemple d'une ligne de données
CIVILITE_ENSEIGNANT Monsieur
NOM_ENSEIGNANT DUPONT
PRENOM_ENSEIGNANT Jean
ADRESSE_ENSEIGNANT 38 av de chanzy 53000 Laval
TELEPHONE1_ENSEIGNANT 02-20-40-53-23
TELEPHONE2_ENSEIGNANT 06-77-40-53-23
EMAIL_ENSEIGNANT jdupont@yahoo.fr
DISCIPLINE_ENSEIGNANT Informatique
